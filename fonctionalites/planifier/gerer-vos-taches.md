# Créez votre plan d'action

## Pour créer une tâche, deux possibilités :

* Soit vous créez une tâche directement dans le plan d'action
* Soit vous créez une tâche directement dans un autre module de l'application \(registre de traitement, notifications violation de données, cookies...\)

### Créer une tâche dans le plan d'action

Dans la page principale du module "planification", cliquez sur "nouvelle tâche". Une nouvelle fenêtre "Nouvelle tâche" apparaît à droite de l'écran vous permettant de décrire la tâche désirée. Une fois décrite, cliquez sur sauvegarder.
