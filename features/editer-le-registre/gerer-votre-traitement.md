# Design and manage your processing

## What is a data processing?

A processing activity is an operation or set of operations carried out on personal data, such as payroll management. All the processing activities taken together constitute the record of processing.

## Create a processing activity

To create a processing activity, first click on "New data processing" in the Record section.

![](../../.gitbook/assets/image%20%28141%29.png)

Fill in the description of the processing activity and you will be taken to the questionnaire section where you can add all the required information.

## Modify a data processing activity 

To modify an existing processing activity, click on the "Edit" button linked to the processing:
